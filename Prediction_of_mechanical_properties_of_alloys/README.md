# Прогнозирование механических свойств сплавов на основе их состава

Проект направлен на разработку модели машинного обучения для прогнозирования ключевых механических свойств металлических сплавов, таких как прочность, пластичность и коррозионная стойкость. Модель использует данные о химическом составе сплавов, условиях обработки и микроструктурных характеристиках для построения регрессионных и классификационных прогнозов.

---

## Цели проекта
- Построение регрессионных моделей для прогнозирования свойств сплавов.
- Определение оптимальных типов сплавов под заданные эксплуатационные условия.
- Выявление ключевых факторов, влияющих на механические свойства материалов.
- Автоматизация пайплайна обработки данных и прогнозирования.

---

## Структура проекта

### Данные
- Источники данных: Materials Project Database, NIST Materials Data Repository, Kaggle Alloys Properties Dataset, Mendeley Data.
- Типы данных: химический состав, параметры термообработки, микроструктурные характеристики.

### Предобработка данных
- Очистка от пропущенных значений и дубликатов.
- Нормализация числовых признаков (концентрация элементов, температура).
- Кодирование категориальных переменных (типы обработки).

### Анализ и визуализация
- Исследование корреляций между составом сплава и его свойствами.
- Построение тепловых карт (heatmap) для выявления значимых факторов.

### Моделирование
- Регрессионные модели: Random Forest, Gradient Boosting.
- Классификация сплавов по эксплуатационным параметрам.
- Валидация с использованием кросс-валидации и метрик (MAE, R²).

### Интерпретация результатов
- Анализ важности признаков с помощью SHAP и LIME.
- Рекомендации по оптимизации состава сплавов.

### Визуализация
- Интерактивный дашборд в Tableau/Power BI для анализа прогнозов и трендов.

---

## Особенности проекта
- **Совмещение CNN и MLP**: анализ изображений микроструктур и метаданных (температура, давление).
- **Transfer Learning**: использование предобученных моделей Vision Transformer.
- **Автоматизация**: реализация пайплайна обработки данных с Apache Airflow.

---

## Инструменты
- **Языки программирования**: Python (Pandas, Scikit-learn, Matplotlib/Seaborn).
- **Базы данных**: SQL для управления структурой данных.
- **Визуализация**: Tableau, Power BI (TO-DO)
- **ML-фреймворки**: TensorFlow/Keras для работы с нейронными сетями.
