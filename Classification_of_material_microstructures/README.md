# Классификация микроструктур материалов по изображениям SEM/TEM

Проект направлен на автоматизацию анализа микроскопических изображений для определения типа кристаллической структуры и выявления дефектов в полимерах и металлах. Решение предназначено для контроля качества в производстве композитных материалов, что позволяет оптимизировать процессы и снизить долю брака.

## Особенности проекта
- **Автоматизированная классификация**: Использование моделей глубокого обучения для точного определения типов микроструктур.
- **Интеграция в производственные системы**: Готовое решение для промышленного внедрения через REST API.
- **Интерпретируемость результатов**: Визуализация областей интереса с помощью Grad-CAM.

## Технологии
- **ML-фреймворки**: PyTorch, TensorFlow.
- **Модели**: ResNet, EfficientNet с дообучением (fine-tuning).
- **Предобработка изображений**: OpenCV, U-Net для сегментации.
- **Веб-интерфейс**: Flask/Django.
- **Развертывание**: Docker, AWS/GCP.

## Этапы реализации
1. **Сбор данных**:
   - Основные датасеты: Capillary Thinning Analysis (UMN), Holey Substrate-Directed Strain Patterning.
   - Дополнительные источники: открытые данные из журнала *Materials Chemistry and Physics*.
2. **Предобработка**:
   - Аугментация (повороты, изменение контраста).
   - Сегментация областей интереса.
3. **Обучение модели**:
   - Использование предобученных CNN.
   - Настройка гиперпараметров и валидация.
4. **Интеграция и развертывание**:
   - Веб-интерфейс для загрузки изображений.
   - REST API для взаимодействия с промышленными системами.
   - Контейнеризация и облачное развертывание.
